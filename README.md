# JQL_Revisiting_the_MAL

This repository provides the supporting data for the manuscript: **Revisiting the Menzerath-Altmann Law Across Nine Indo-European Languages and Four Registers**.

## Folder 1

Folder 1 contains the following:

1. Absolute frequencies, relative frequencies, and cumulative relative frequencies of sentence length and paragraph length;
2. The corresponding word-token counts (excluding punctuation), finite-verb token counts, mean sentence length, and mean clause length;
3. The corresponding fitting results.

## Folder 2

Folder 2 contains 2% paragraph samples extracted from the annotated versions of the following nine corpora:

sqGLOBE, hyGLOBE, ltGLOBE, gaGLOBE, daGLOBE, deGLOBE, urGLOBE, itGLOBE, and frGLOBE.

## Folder 3

Folder 3 contains the top-2,000 word-frequency list for each corpus.
